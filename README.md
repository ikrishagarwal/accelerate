# Accelerate

A collection of interactive physics simulations built with p5.js, designed for educational purposes.

[Live Site](https://ikrishagarwal.github.io/accelerate/)

# Simulations

## Double Pendulum

An interactive double pendulum simulation that demonstrates chaotic motion. Users can adjust parameters such as mass and length of the pendulums, and observe the resulting motion.

[Live Demo](https://ikrishagarwal.github.io/accelerate/pendulum.html)

### Features

- You can export the trail design of the double pendulum.
- There's music as the pendulum flows along it's path.
